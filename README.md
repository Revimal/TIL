### TIL에 정리할 내용
* 진짜로 오늘 새로 배우게 된 사실들
* 공부한지 오래되어 잊혀져가다 다시 생각나게 된 기술들

* 슬슬 머릿속 기억만으로는 버티기 버거워지기 시작함

* 글로 정리할 수 있는 지식이 진짜로 내가 이해한 지식이라는 생각도 들어서
